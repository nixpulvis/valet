Expand description

Types that serialize to a bitcode + snap-compressed byte buffer and seal under a Key<T> via AES-GCM-SIV.

The type parameter T is the key’s scope marker; it carries no data and only constrains which Key<T> this stash will accept at the type level.

Typical use:

ⓘ
#[derive(Encode, Decode)]
struct Foo { /* ... */ }

impl Stash<Lot> for Foo {}

let foo = Foo { /* ... */ };
let sealed = foo.encrypt_with_aad(lot.key(), b"aad")?;
let foo2 = Foo::decrypt_with_aad(&sealed, lot.key(), b"aad")?;
