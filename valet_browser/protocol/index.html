valet_browser

Module protocol 

Source
Expand description

Native-messaging client bridge used by the popup and content scripts.

Both sides build typed Call payloads and issue them through NativeMessageClient, a Handler that frames each request as a base64 bitcode string and forwards it via browser.runtime.sendMessage to the background script. The background pumps the bytes to the native host over its shared valet::protocol::NativeMessageClient.

Typical use:

ⓘ
let list = protocol::NativeMessageClient.call(ListUsers).await?;
